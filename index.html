LC101 Home

learn.launchcode.org is the home of content for LaunchCode's LC101 class. You must be enrolled in an active section of LC101 to access the site.

Coding Prep

Unit 0

The Coding Prep unit includes topics that are essential for learning to code: math, file management, and command line usage. Shoring these up will set up students for success.

Think Python

Unit 1

Think Python introduces students to fundamental concepts in procedural programming along with object-oriented basics. It is based on the open-source textbook used by dozens of world-class universities around the world.

Web Fundamentals

Unit 2

The Web Fundamentals unit of LC101 covers fundamental concepts for building full-stack web applications.

Back-end Java

Unit 3 / Skill Track

The Back-end Java skill track teaches essential object-oriented concepts in Java, as well as how to build sophisticated web applictions using the Spring Boot Framework

Back-end C#

Unit 3 / Skill Track

The Back-end C# skill track introduces Microsoft's C# programming language, along with the basics of building web applications with the ASP.NET Core framework.

Front-end Web

Unit 3 / Skill Track

The Front-end Web skill track dives into the wild and fun world of front-end web developmet, covering HTML, CSS, and Javascript.
